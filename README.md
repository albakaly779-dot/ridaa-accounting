# Ridaa Accounting - MVP Scaffold (RTL Arabic)

تشغيل مبدئي للمشروع المحلي:

1. انسخ `.env.example` إلى `.env` في مجلد `backend` ثم عدّل إعدادات قاعدة البيانات إن لزم.
2. شغّل:
   ```
   docker-compose up --build
   ```
3. واجهة الـ Frontend: http://localhost:3000
   API backend: http://localhost:8000

هذا سكفولد مبدئي يحتوي ملفات جاهزة لتطوير تطبيق Laravel + Next.js بنمط RTL.
